# Omid's Blog

This is my personal blog where I share my thoughts and experiences about technology, programming, and other interesting topics.

## About

This blog is built using [Hugo](https://gohugo.io/) and is based on the [lilianweng.github.io](https://github.com/lilianweng/lilianweng.github.io) theme.

## Features

- Clean and modern design
- Responsive layout
- Support for code highlighting
- Tag-based organization
- Search functionality
